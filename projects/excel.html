Excel Marketing Funnel Analysis

Conversion rate optimization dashboard built in Excel to track CTR, stage transitions, and revenue distribution.

View GitHub Repository

Project Overview

This project focuses on analyzing user journey data through a marketing funnel using Excel to identify drop-off points and optimize conversion performance.

Objective

To evaluate conversion rates across funnel stages and measure the impact of marketing performance on revenue generation.

Dashboard Preview

Tools & Technical Implementation

  • Microsoft Excel
  • Advanced Formulas (IF, SUMIFS, INDEX-MATCH)
  • Pivot Tables & Charts
  • Conditional Formatting
  • Automated KPI Calculations

Metrics Tracked

  • Click-Through Rate (CTR)
  • Stage-to-Stage Conversion Rate
  • Customer Acquisition Analysis
  • Revenue Split (New vs Returning Customers)

Business Impact

Analysis revealed the largest drop-off occurred between click and add-to-cart stages, highlighting opportunities for UX optimization and targeted remarketing strategies.
